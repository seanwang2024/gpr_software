# DZX 参数逆向实验说明

## 实验目的

本实验包含 **17 个 DZX 配置文件**（`PROJ__001.DZX` ～ `PROJ__017.DZX`），采用**控制变量法**进行参数逆向分析。

除当前实验项外，其余参数均保持一致，以便准确定位 DZX 文件中各参数对应的 XML 配置字段。

---

# 基准参数（默认配置）

除表中说明的修改项外，其余参数均采用以下默认设置：

| 参数 | 默认值 |
|------|--------|
| 增益模式 | 自动 |
| 增益点数 | 3 点 |
| FIR 高通 | 关闭 |
| FIR 低通 | 关闭 |
| FIR 叠加 | 关闭 |
| FIR 背景去除 | 0 |
| IIR 高通 | 800 MHz |
| IIR 低通 | 100 MHz |
| IIR 叠加 | 0 |
| IIR 背景去除 | 0 |
| 信号底跟踪 | 关闭 |

> **说明：**
>
> - **关闭**：表示该功能未启用。
> - **0**：表示参数值为 0（并非关闭开关，而是对应功能的默认数值）。

# 参数说明

| 参数 | 说明 |
|------|------|
| 增益模式 | 自动增益或手动增益。 |
| 增益点数 | 自动或手动增益控制点数量。 |
| 手动增益值 | 各增益控制点对应的增益值，仅手动模式有效。 |
| FIR 高通 | FIR 高通滤波截止频率。 |
| FIR 低通 | FIR 低通滤波截止频率。 |
| FIR 叠加 | FIR 滑动叠加（平滑）次数。 |
| FIR 背景去除 | FIR Background Removal 参数值。 |
| IIR 高通 | IIR 高通滤波截止频率。 |
| IIR 低通 | IIR 低通滤波截止频率。 |
| IIR 叠加 | IIR 滑动叠加（平滑）次数。 |
| IIR 背景去除 | IIR Background Removal 参数值。 |
| 信号底跟踪 | 是否启用 Signal Floor Tracking（信号底跟踪）。 |
