## 附录:版本变更记录

> 每个新版本发布时,在此追加该版本的变更摘要和新增/修改的测试用例编号。

### v1.0.25 (2026-07-16)

**功能清单**:
- ribbon 三级菜单(开始 / 数据处理 / AI 处理)
- DZT 文件打开(按钮 + 拖拽)、保存、关闭、文件头查看
- 多标签页 + 水平/垂直选项卡组 + 拖拽移动 + 右键菜单
- 图像缩放(水平放大/缩小/还原/堆积图/调色板)
- 简易处理:一键处理、调节零点、校正零偏、背景消除、调节增益、数字滤波、批处理
- 数据处理:零点调节、滤波(校正零偏/背景消除/数字滤波)、其他处理(滑动平均/道间均衡/数学运算/反褶积/希尔伯特/克希霍夫偏移/一键处理)
- AI 处理:AI 识别(YOLOv8 三分类)、保存 JPG、生成 PDF 报告
- 在线升级:版本检查、下载(圆形放大效果)、二选一(立即重启/下次启动)
- welcome 页:比例铺满、4 图标悬停放大 + 功能说明
- 安装包:Inno Setup 简体中文、劳雷 logo 图标、桌面/开始菜单快捷方式
- 标尺:时间(ns,自适应刻度)、深度(m,按介电常数换算)、道号
- DZX XML 配置文件解析(uuencode BinaryData,11 种 Process 类型)

**测试用例范围**: TC 1.1 ~ 11.2(共 55 项)
